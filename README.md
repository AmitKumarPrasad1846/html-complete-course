# HTML Complete Course 🚀

Complete HTML course for beginners to advanced learners. Includes practical examples, exercises, and detailed notes.

## 📁 Course Structure

- **1_How_website_work** - How websites work (client-server, DNS, HTTP)
- **2_your_first_html_website** - Create your first HTML page
- **3_Basic_structure_html_website** - HTML document structure
- **4_heading_paragraph_link** - Headings, paragraphs, links
- **5_Image_audio_video_list_tables** - Media elements, lists, tables
- **6_SEO_and_core_web_vitals_in_HTML** - SEO basics, Core Web Vitals
- **7_Form_input_tag** - Forms and input elements
- **8_Inline_block_element** - Inline vs block elements
- **9_ID_Classes** - IDs and classes
- **10_video_audio_media** - Advanced video/audio
- **11_semantic_tag_html** - Semantic HTML tags
- **12_exercise_1** - Practice exercise
- **13_entities_code_tag_html** - HTML entities, code tags
