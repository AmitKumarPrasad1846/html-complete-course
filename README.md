<ol>

  <li>Select <b>"Download ZIP"</b></li>
  <li>Extract the ZIP file</li>
  <li>Open any folder and double-click HTML files</li>
</ol>

<h3>Method 2: Using Git (Recommended)</h3>

<pre>
<b>Step 1:</b> Clone the repository
<code>git clone https://github.com/AmitKumarPrasad1846/html-complete-course.git</code>

<b>Step 2:</b> Navigate to the directory
<code>cd html-complete-course</code>

<b>Step 3:</b> Start with Module 1
<code>cd "1_How_website_work"</code>

<b>Step 4:</b> Open in browser
<code># Simply double-click any .html file
# OR use Live Server in VS Code</code>
</pre>

<h3>For VS Code Users (Best Experience)</h3>

<ol>
  <li>Install <b>Live Server</b> extension from marketplace</li>
  <li>Open any HTML file</li>
  <li>Right-click and select <b>"Open with Live Server"</b></li>
  <li>Your browser will open with live reload!</li>
</ol>
