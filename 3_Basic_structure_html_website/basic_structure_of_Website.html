PRE Tag (Preformatted Text)

Preserves spaces and new lines as written

<pre>
  This is
  pre-formatted
</pre> 
This is
  pre-formatted
    text with
      preserved spacing

📝 Note: Unlike normal HTML, the PRE tag respects every space and newline exactly as typed in the code.
