From Chapter 0: Introduction to HTML

🔰 What is HTML?

• HTML stands for Hyper Text Markup Language
• It is the foundation of every website on the internet
• Used to define the structure and layout of web pages

🎨 Why Do We Need CSS and JavaScript?

Layer Purpose
HTML Defines structure (skeleton)
CSS Adds style (color, layout)
JavaScript Adds interactivity (logic)

🚗 Beautiful Analogy:

• HTML = Car body (metal frame)
• CSS = Paint, seat covers, lights (design)
• JavaScript = Engine, gear system (functionality)
